SPAN 100

An introductory Spanish class intended for students that took a Spanish class in high school but did not perform well enough on the Spanish entrance exam to place out of first-level Spanish courses. This course provides the critical foundations for learning the Spanish language and the groundwork on which future UNC Spanish classes build upon.
